## Setup Directions
------

1. ```npm install```
2. ```composer install```
3. Set database infos in the .env
4. ```php artisan migrate:fresh --seed```
5. ```php artisan serve```
6. ```npm run hot```
7. ```Goto: http://localhost:8000 or your APP_URL (in the .ENV)```
